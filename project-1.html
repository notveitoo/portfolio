Intelligent Room Access Using Computer Vision System

Intelligent Room Access Using a Computer Vision System enhances security, time logging, and convenience for instructors. This system allows instructors to lock/unlock rooms using facial recognition, eliminating the need for physical keys and ensuring room availability checks can be done remotely.

Project Overview

This system is designed to manage all functions of a college admin. It allows the admin to monitor activities within the building and generate reports on room usage.
